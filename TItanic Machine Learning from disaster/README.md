# Titanic Survival Prediction

A machine learning project that predicts passenger survival on the Titanic using classification algorithms.

## Dataset

The dataset is from [Kaggle's Titanic Competition](https://www.kaggle.com/c/titanic).

## Project Structure

```
├── dataset/
│   ├── train.csv
│   ├── test.csv
│   └── submissions/
├── models/
├── notebooks/
│   └── notebook.ipynb
└── README.md
```

## Models

- Logistic Regression
- Random Forest Classifier
- XGBoost Classifier

## Usage

Open and run `notebooks/notebook.ipynb` to train models and generate predictions.

numpy
